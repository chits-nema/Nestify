# Nestify

A beautiful, modern web application to help users find their forever home.

## Features

- **Intuitive Search**: Search homes by location, type, or features
- **Advanced Filters**: Filter by home type, number of bedrooms, and price range
- **Interactive Listings**: View detailed information for each property
- **Favorites System**: Save your favorite homes with a simple click
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Beautiful gradient designs and smooth animations

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Python 3 (optional, for local testing with HTTP server)

### Running the Application

1. Clone this repository
2. Open `index.html` in your web browser, or
3. Start a local server:
   ```bash
   python3 -m http.server 8000
   ```
   Then navigate to `http://localhost:8000/index.html`

## Project Structure

- `index.html` - Main HTML structure
- `styles.css` - CSS styling and responsive design
- `script.js` - JavaScript functionality for search, filters, and interactivity

## Features in Detail

### Search Functionality
Users can search for homes by entering location names, home types, or features in the search bar.

### Filter Options
- **Home Type**: Filter by House, Apartment, Condo, or Townhouse
- **Bedrooms**: Filter by minimum number of bedrooms (1+, 2+, 3+, 4+)
- **Max Price**: Filter by maximum price range

### Home Listings
Each listing displays:
- Price
- Title and description
- Location
- Number of bedrooms and bathrooms
- Square footage
- Favorite toggle button

### Responsive Design
The website is fully responsive and adapts to different screen sizes for optimal viewing on all devices.

## Technologies Used

- HTML5
- CSS3 (with modern features like Grid, Flexbox, and animations)
- JavaScript (ES6+)
- No external dependencies - vanilla JavaScript only!
